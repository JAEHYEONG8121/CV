1. Feature Detection 및 Description 방법 변경
  - SIFT 사용
  - SURF, ORB 알고리즘 사용해보기

2. Matcher 알고리즘 변경
  - Matching방법 다양하게 해보기
  - BFMatcher : 모든 특징점 간의 거리를 계산하므로 더 정확한 매칭 가능 but, 속도 저하

3. RANSAC 파라미터 조정
  - 임계값이 5.0을 조정 -> 특정 장면에 적합한 값으로 변경해보기
  - 데이터에 따라 임계값의 어떤 차이를 띄는지 분석

4. 블렌딩 방식 변경
  - 단순히 두 이미지를 겹쳐서 블렌딩하는 방식을 사용했음
  - But, 이미지가 매우 다를 경우, 경계부분에서 가시적인 경계선이 발생할 수 있음
  - 더 부드러운 블렌딩 방식 해보기
  - feather blending, multi-band blending등을 고려해보기
  - cv2.seamlessClone() 함수를 통해 다양한 블렌딩 방식을 제공하고 있음
  - 실험을 통해 어떤 블렌딩 방식이 가장 적합한지 확인해보기
